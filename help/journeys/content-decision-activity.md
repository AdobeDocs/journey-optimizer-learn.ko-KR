# 콘텐츠 결정 활동

여정 캔버스에서 콘텐츠 의사 결정 활동을 통해 사용자가 개인화된 콘텐츠 오퍼에 AJO Decisioning을 활용하는 방법을 알아봅니다. 이 기능을 사용하여 검색된 오퍼를 기반으로 프로필을 라우팅하거나 외부 시스템에 오퍼를 전송하여 여정의 효율성과 개인화를 향상시킵니다.
