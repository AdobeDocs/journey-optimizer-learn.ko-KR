# 사용 사례

Adobe Journey Optimizer에서 제공하는 실제 마케팅 시나리오를 살펴보십시오. 이러한 사용 사례에서는 브랜드가 온보딩 및 재참여부터 개인화된 오퍼 및 크로스 채널 오케스트레이션에 이르기까지 보다 스마트한 고객 여정을 구축하는 방법을 보여줍니다. 각 예제는 전략을 실제 실행과 연결하여 실제 비즈니스 목표에 제품 기능을 적용하는 데 도움이 됩니다.
