# 태그 만들기

[!DNL Offer Decisioning]에서 [태그를 만드는 방법](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/create-components/creating-tags.html?lang=ko)을 알아봅니다. 태그는 오퍼의 선택적 구성 요소입니다. 오퍼를 구성하고 다이내믹 컬렉션에서 그룹화하는 데 사용할 수 있습니다.
