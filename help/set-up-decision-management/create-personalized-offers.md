# 개인화된 오퍼 만들기

[!DNL Offer Decisioning]에서 개인화된 오퍼를 만드는 방법을 알아봅니다. 개인화된 오퍼에는 관련 고객에게만 표시하는 데 도움이 되도록 해당 오퍼와 연관된 자격 규칙이 있습니다. 예를 들어, 겨울 동안에만 눈이 내리는 지역에 대해 눈놀에 대한 오퍼를 제한할 수 있습니다.
