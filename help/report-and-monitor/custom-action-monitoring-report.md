# 사용자 정의 액션 모니터링

사용자 지정 작업을 통해 여정에서 서드파티 엔드포인트로 전송되는 HTTP 호출의 작동 상태 및 성능을 효과적으로 모니터링하는 방법을 알아봅니다. HTTP 오류, 지연 병목 현상 및 게재에 영향을 줄 수 있는 최대 가용량 문제에 대한 가시성을 확보합니다.

이 보고서는 문제를 식별하고 여정 성능을 향상시키며 원활하게 작동하는 데 도움이 됩니다.
