# Decisioning을 사용하여 이메일 개인화

이 튜토리얼에서는 Adobe Journey Optimizer Decisioning을 사용하여 개인화된 이메일 오퍼를 보내는 실제 사용 사례를 살펴봅니다. 시나리오는 간단합니다. 연간 수입 및 충성도 상태와 같은 프로필 속성을 기반으로 고객에게 타겟팅된 이메일 오퍼를 보냅니다.
자습서 전체에서 단계별로 진행하여 Adobe Experience Platform에서 프로필을 만드는 단계부터 이메일 채널을 구성하는 단계 및 의사 결정을 사용하여 가장 관련성이 높은 오퍼를 자동으로 선택하는 캠페인을 구축하고 실행하는 단계까지 진행합니다.

## 이 자습서의 사전 요구 사항

* Experience Platform 액세스

* Experience Platform 개념(프로필, 대상, 데이터 세트)에 대한 기본 이해

* Journey Optimizer 친숙도
