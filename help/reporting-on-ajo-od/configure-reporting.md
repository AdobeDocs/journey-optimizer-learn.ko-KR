# 보고 데이터 소스 설정

Adobe Journey Optimizer Offer Decisioning에 대한 보고 설정을 시작하려면

- Journey Optimizer에 로그인

- _**관리 ->구성 ->보고 ->관리 ->데이터 세트 추가**_(으)로 이동
- 오퍼 노출을 저장하고 이전 자습서에서 생성된 이벤트를 클릭하는 _**날씨 스키마 데이터 세트**_&#x200B;를 선택하십시오.
