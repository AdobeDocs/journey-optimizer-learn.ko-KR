# 양식 제출 시 여정 트리거

이 사용 사례에서는 고객 양식 제출이 Adobe Journey Optimizer 여정을 실시간으로 트리거하여 사용자가 제공한 정보를 기반으로 즉시 개인화된 후속 커뮤니케이션을 활성화하는 방법을 보여 줍니다.


## 이 자습서의 사전 요구 사항

* Experience Platform 액세스

* Experience Platform 개념(프로필, 대상, 데이터 세트)에 대한 기본 이해

* Journey Optimizer 친숙도

## 구현 단계

* Adobe Experience Platform에서 이벤트 스키마 정의
* Adobe Journey Optimizer에서 사용자 지정 이벤트 구성
* 이메일 채널 구성
* 사용자 지정 이벤트에 의해 트리거된 여정 빌드
* Adobe Experience Platform으로 양식 제출 데이터 보내기
* 이메일에 참조 데이터를 포함하도록 데이터 세트 조회 구성
