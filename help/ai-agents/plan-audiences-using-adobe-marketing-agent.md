# Adobe Marketing Agent for Microsoft 365 Copilot을 사용하여 대상자 계획

Adobe Marketing Agent for Microsoft 365 Copilot이 Microsoft Teams 내에서 직접 대상 계획을 간소화하는 방법에 대해 알아봅니다. 이 도구를 사용하여 마케터는 플랫폼을 전환하거나 운영 팀에 요청을 제출하지 않고도 충성도 대상을 식별하고, 여정 계보를 확인하고, 캠페인 후보의 유효성을 검사할 수 있습니다.

이 비디오에서는 캠페인을 위한 대상을 선택, 분석 및 준비하는 프로세스를 보여 줍니다. 이 비디오에서는 에이전트가 실행 가능한 통찰력을 생성하고, 관련 제안을 표면화하고, 대상과 여정 간에 계보 시각화를 제공하는 방법을 보여 줍니다. 마케터는 데이터를 효율적으로 관리하고 워크플로우를 간소화하여 집중 타기팅을 보장하고 캠페인 결과를 향상시킬 수 있습니다.

아래 포함된 비디오를 통해 Adobe Marketing Agent의 라이브 데모 버전을 확인해 보십시오.

>[!VIDEO](https://video.tv.adobe.com/v/3491171/?learn=on&enablevpops)
