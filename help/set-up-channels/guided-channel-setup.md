# 안내형 채널 설정

안내형 채널 설정을 사용하면 마케팅 팀이 [캠페인] 및 [여정] 작성을 시작할 수 있도록 Experience Platform, Journey Optimizer, 데이터 수집에 흩어져 있는 리소스 중 필요한 것을 사용하여 웹 및 모바일 채널을 신속하게 설정하고 확인할 수 있습니다.
