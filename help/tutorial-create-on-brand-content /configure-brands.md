# Adobe Journey Optimizer에서 브랜드 구성

이 비디오에서는 브랜드 가이드라인 문서를 업로드하고, 주요 브랜드 요소를 구조화하고, 작성 스타일, 시각적 컨텐츠 및 ID 세부 사항을 정의하는 방법을 안내합니다. 또한 이러한 지침을 세분화하고 사용자 정의하여 AI가 생성한 컨텐츠와 자동화된 품질 검사를 지원하여 모든 채널에서 일관된 브랜드 내 커뮤니케이션을 보장하는 방법을 살펴볼 수 있습니다.
