# Adobe Journey Optimizer에서 브랜드 구성

모든 채널에서 일관되고 브랜드 내 커뮤니케이션을 보장하기 위해 Adobe Journey Optimizer에서 브랜드 지침을 구성하는 방법을 알아봅니다. 이 안내서에서는 스타일, 시각적 개체 및 ID 세부 사항을 포함하여 브랜드 문서를 업로드하고 주요 브랜드 요소를 구성하는 방법에 대해 설명합니다. 또한 이러한 지침을 세분화하여 콘텐츠 생성을 강화하고 품질 검사를 자동화하여 브랜드 정체성을 강하고 효과적으로 유지하는 방법을 살펴볼 수 있습니다. 단계별 연습은 아래 비디오를 참조하십시오.
