# AI를 사용하여 브랜드 내 이메일 콘텐츠 만들기

Adobe Journey Optimizer의 AI 기능을 통해 대규모로 온브랜드 이메일 콘텐츠를 빌드하는 방법을 살펴볼 수 있습니다. 이 연습에서는 브랜드 가이드라인과 참조 콘텐츠를 사용하여 생성을 안내하고, 생성 모델을 선택하고, AI에서 생성한 텍스트와 이미지를 템플릿에 직접 삽입하는 단일 프롬프트에서 생성된 전체 이메일을 보여 줍니다. 또한 동일한 브랜드 자산은 자동화된 품질 검사를 지원하여 규정 준수, 색조 및 브랜드 정렬에 대한 각 변형의 점수를 매깁니다.
