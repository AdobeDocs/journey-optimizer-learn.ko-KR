# 웹 푸시 알림 만들기

웹 푸시를 사용하면 브라우저가 백그라운드에 있더라도 웹 사이트에서 사용자의 장치로 직접 알림을 전송할 수 있습니다. Adobe Journey Optimizer에서 웹 푸시 알림을 만들고 전송하는 방법을 알아봅니다. 이 튜토리얼에서는 채널 구성 선택, 메시지 작성, 대상 선택 및 캠페인을 게시하여 모바일 장치를 넘어 범위를 확장하는 방법을 안내합니다.
