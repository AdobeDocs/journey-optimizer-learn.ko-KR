# 컬렉션 만들기

컬렉션을 사용하면 환경 설정에 따라 결정 항목을 분류하고 그룹화할 수 있습니다. 이러한 카테고리는 결정 항목의 속성을 활용하는 규칙을 작성하여 생성됩니다.

* Journey Optimizer에 로그인
* 의사 결정 -> 카탈로그 -> 컬렉션 ->컬렉션 만들기
* 화면에 표시된 대로 컬렉션 이름과 컬렉션 규칙을 지정합니다
